ENCONTRO COM DEUS - SITE PRONTO V3

O site está pronto para celular e desktop.

AJUSTES RÁPIDOS:
1. Abra o arquivo config.js
2. Preencha googleFormsUrl com o link público do Google Forms
3. Altere eventDate com a data oficial do encontro
4. Publique a pasta inteira em qualquer hospedagem estática

ARQUIVOS PRINCIPAIS:
- index.html -> layout do site
- config.js -> link do formulário e data do evento
- assets/ -> logo e fotos

OBSERVAÇÃO:
A data do evento aparece em destaque na primeira dobra do site.
